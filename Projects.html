Scratch Project

Duration: August 30th 2018 - September 7th 2018

For our first project in Computer Science Principles, my partner and I had to choose whether to make a game or to make a story. We went with the game, and created a platformer from scratch over the course of the project. It has multiple levels, with a lives system and 2 mini bosses, with one final boss at the end. There is also a speed run option, which enables a timer to track how fast you can beat the game. Try it out with the embed! Sometimes there can be a glitch where it does not display the correct instructions, but simply restart and it will. Also, if you get the message that it can't be loaded because flash is not running, click the link and allow flash. Project Documentation
